RGB game is a prediction game, that aims to actually be used as a mining game in future.

You have to place your predictions by selecting three numbers, one number from each row, then click run button to match random match.

You can only make predictions between 50 to 52 in an overall game. at the end of each game your earned score is converted to naira that you can cash out or use to purchase items or services on our site. you can start another game instantly

click Run to start a run session(RS) There are three run session per restart session(RTS) there is 50 to 51 RTS in a Game New session (GNS).


Step 1: Click run 3 times
Step 2: Click restart 1 time Repeat until you hit run for about 50 to 51 times
Step 3: Game New Session will start
