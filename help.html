RGB game is a prediction game, that aims to actually be used as a mining game in future.

You have to place your predictions by selecting three numbers, one number from each row, then click run button to match random match.

You can only make predictions between 50 to 52 in an overall game. at the end of each game your earned score is converted to naira that you can cash out or use to purchase items or services on our site. you can start another game instantly

G-Session(Game Session) of the game is run three times after which you restart another B-session(Begining Session) by clicking restart button.

The game ends ofter game played reach between 50, 51 or 52 the another R-Session(Restart Session) start.

If thetotal score is more than 50 you will earn naira else you will not earn anything.

If the total scores is more than 50 then 50 will be deducted from the total score the remaing scores and the remaining scores will be multiply by 10 naira. what ever you it is will be the total mined naira you earn per R-Session and it is equivalent to Naigerian Naira.

You can cash it out, trade it for real money or crypto currencies or use it to purchase some valuable thing online<> Back to Game

THINGS TO WATCH

Total score earn is the first red button ontop

Total naira earn is prefix with the sign (#)

Prediction button is automaticaly set to 1, 1, 1 for each row, if you did not select ang prediction

The button above the run button indicate total score for each row first figure indicate score for first row, second figure indicate score for second row third figure indicate score for the last row.

Restart button is click after Run Button have been click three times

GamePlayed board display total played in the overall game

SessionCount board display number of present session of the game

SessionScore Board display total score per session
